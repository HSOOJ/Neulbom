## **프로젝트 소개**

### 1️⃣ 주제

- 서비스 내용

  ```
  똑똑한 건강 관리 앱, 늘봄

  고혈압, 당뇨 등 만성 질환을 가진 분들의 건강 관리를 도와드립니다!
  혈당, 혈압을 주기적으로 기록하고,
  음식 사진을 찍으면 알아서 영양 정보를 계산해주고 기록해줍니다.
  권장 섭취량을 기준으로 과도한 나트륨, 당 섭취를 예방합니다.
  기록한 식단을 기반으로 전문가 상담을 제공합니다.
  ```
