## 🌳 Branch Naming Rule

- develop / (`FE` or `BE` )-`유형(feat, fix 등)`브랜치를 사용한다.

- Merge Request
  - 브랜치 이름: (`FE` or `BE` )-`유형(feat, fix 등)`/ `jira 이슈번호`-`기능 요약`
- `develop` : (`FE` or `BE` ) - `유형` / `jira 이슈번호`-`기능 요약`

- 예시
  `git branch FE-feat/#123-user-login`
